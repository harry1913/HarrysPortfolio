Crypto4CRFID

Introduction This project aimed to build a secure environment between an RFID reader and CRFID with Hash function.

Crypto4CRFID 프로젝트는 새로운 RFID 중 하나인 Computational RFID (CRFID) 과 Hash function을 이용한 Secure RFID 통신 환경을 만들어내는 프로젝트입니다. 본 프로젝트는 총 4명의 학생으로 구성되었고 그 중에서 CRFID 와 RFID Reader 간의 데이터 송/수신, Hash function 암호와 복호화를 담당하는 응용프로그램을 만드는데 기여하였습니다. 해당 응용 프로그램은 Python과 Low-level reader protocol (LLRP) 그리고 웹 서버를 이용하여 RFID data를 읽기/쓰기 가 가능하였고 User는 간단한 Bootstrap GUI를 통해서 RFID data 읽기/쓰기를 제어할 수 있도록 구축되었습니다. CRFID는 자체적으로 FRAM을 가지고 있어 기존의 RFID Memory bank 보다 훨씬 많은 양의 데이터를 저장하도록 되어있습니다. 이 프로젝트에서 해당 응용프로그램은 FRAM에 직접으로 접근하여 데이터를 읽기 또는 쓰기가 가능하도록 되어있습니다. 최종적으로 응용프로그램은 User에게 GUI를 제공함으로써 CRFID의 데이터를 쉽게 읽기 및 쓰기가 가능하지만 고유의 CRFID의 Hash function 값과 응용프로그램에서 CRFID firmware byte code를 Hash function 값으로 생성한 값과 일치하면 데이터의 접근 및 호출이 가능하도록 하였습니다.
Technique Python, C++, LLRP, WISP5, CRFID.
