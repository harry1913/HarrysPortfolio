Global Tax Free System

Introduction Building application software that can help travellers to refund tax.

Global Tax Free (GTF) 차세대 시스템 구축은 현재 운영 중인 GTF회사의 차세대 시스템을 구축하는 프로젝트였습니다. GTF는 외국인 여행자를 대상으로 여행국에서 물건을 구매하고 의무적으로 지불해야하는 부가세에 대한 환급을 도와주는 회사입니다. 해당 회사의 고객들의 영수증 및 환급 처리를 위한 차세대 시스템 구축에 기여하였습니다. 그 중에서도 GTF에 가입되어 있는 매장에서 환급을 쉽게 도와줄 수 있도록 하는 매장내 응용프로그램을 개발을 맡았었습니다. 해당 프로그램은 기본 Java 언어로 만들어 졌으며 응용 프로그램의 GUI 구축을 위해서 Java Swing을 사용하여 전체적인 틀과 내부의 Data table을 구축하여 고객과 매장 업주간의 신속한 텍스 환급 처리를 도와주는 프로그램입니다. 해당 응용 프로그램은 GTF 서버와 database 연동으로 좀 더 쉽게 고객들의 품목을 관리하도록 하였으며 해당 응용 프로그램은 바코드 및 QR 리더기를 이용하여 좀더 빠르게 고객들은 영수증 내용을 입력이 가능하고 영수증 프린터기와 연동으로 환급영수증을 출력할 수 있도록 프로그래밍 되었습니다.
Technique Java swing, Spring, Javascript, SQL
Period 2013-03 to 2013-09
