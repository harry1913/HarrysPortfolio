Auto PDS Cooperated Development

Introduction Building a web site to share equipment designs with overseas suppliers.

AutoPDS 협업개발은 현대에서 제작된 설계도면을 해외 지사들과 공유하는 시스템입니다. AutoPDS 시스템은 기본적으로 JSP를 배경으로 하며 설계도를 업로드하고 다운로드 할 수 있는 모듈 설치와 정보 공유에 필요한 웹 페이지를 구축하는 일을 담당하였습니다. 프로젝트 짧은 기간 관계상 직접적으로 Active-X 업/다운로드 모듈을 제작할 수 없었기에 Innorix 웹 기반 파일 전송 모듈을 기존 AutoPDS 시스템에 적용하여 시스템을 완료하고 서비스를 제공하였습니다.
Technique Java, JSP, JavaScript.
Period 2011-10 to 2012-04
